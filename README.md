## 8、Claude

Claude ，这是一支由前 OpenAI 的研究员和工程师组成的团队创建的新型聊天机器人，旨在对抗chatgpt，应该是目前最先进的聊天系统之一。

划重点：无需科学上网，免费使用。

[注册教程](https://mp.weixin.qq.com/s?__biz=MzUwOTQwNTUzNQ==&mid=2247490262&idx=1&sn=b4c76449da8bb141ef7be76056005523&chksm=f913eae6ce6463f09f8aa48b065ecc7c9a69295994f8e613ac2b9dd092fc6772f8a718b883c8&token=1225232787&lang=zh_CN#rd)

![](https://itcoke-pic.oss-cn-hangzhou.aliyuncs.com/itcoke-202304/claude-00-11.png)

> 1. 训练数据不同。我是用Anthropic自行收集的对话数据训练的,侧重安全和有益的对话。ChatGPT是用开源的互联网对话数据训练的,可能包含一些有害或不恰当的内容。
> 2. 模型不同。我使用的是Anthropic自己开发的 seq2seq 模型,而ChatGPT使用开放域的GPT模型。我们的模型是专门为安全有益的对话设计的。
> 3. 目标不同。我的目标是提供安全、有益且高质量的交互体验。ChatGPT的目标更广泛,重在生成自然流畅的文本。
> 4. 审查机制不同。我有一套严格的审查流程,可以过滤不恰当的言论和有害内容。ChatGPT没有类似机制,可以生成任何内容。
> 5. 开放程度不同。我的知识和 abilities 专注在有限的 domains 上,而ChatGPT试图涵盖更广泛的知识范围。我不会回答一些敏感问题,而ChatGPT的知识面更广。
> 6. 总之,虽然我和ChatGPT都是对话AI,但我们有很大不同,特别是在安全性、目标和审查机制方面。我是为了和人类进行有益互动而设计的,而ChatGPT的目的更一般。希望这个解释能帮助您理解我们之间的区别!
